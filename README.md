## 📖 About The Project

**Word Counter Pro** is a sleek, modern, and user-friendly web application built with **Python** and **Streamlit** that provides instant text analysis. Whether you're a student working on an essay, a writer crafting an article, or a professional preparing a report, this tool helps you quickly count words, characters, and lines in your text.

The application features a beautiful **dark-themed interface** that's easy on the eyes, making it perfect for extended use. With just one click, you get comprehensive statistics about your text in a clean, organized layout.

## 💻 Installation

Follow these steps to run the app locally on your machine:

### Prerequisites
- **Python 3.7 or higher** installed on your system
- **pip** (install streamlit)

### Step-by-Step Installation

 **Step 1- Clone the repository**
   ```bash
   git clone https://github.com/HaiderNeuralNet/word-counter-app.git
 ```
 **Step 2- Go into the folder**
     ```bash
      cd word-counter-app
     ```
3. **Step 3- Run The App**
   ```bash
streamlit run main.py
Or if you have multiple Python versions:
python -m streamlit run main.py
```
## 📖 How to Use
```
Using Word Counter Pro is incredibly simple:

Step-by-Step Guide
1. Launch the app using the command above

2. Paste or type your text in the large text area

3. Click the 🔍 Count Words button

4. View your  results instantly showing:

>Total Words - Number of words in your text

>Total Characters - Number of characters including spaces

>Total Lines - Number of lines in your text

5. Success message confirms the analysis is complete

6. If text area is empty, a friendly warning will appear
  ```
